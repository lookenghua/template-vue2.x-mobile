# 移动端模板

项目启动

```
yarn install
```

### 开发环境运行
```
yarn serve
```

### 正式环境打包
```
yarn build
```

### 运行单元测试
```
yarn test:unit
```

### 运行端对端测试
```
yarn test:e2e
```

### 静态代码分析和修复文件
```
yarn lint
```
### 支持的功能

1. 默认Vant ui框架,Scss预编译,按需引入
2. 支持px自动转换rem
3. gzip压缩
4. 请求转发
5. 样式重置
